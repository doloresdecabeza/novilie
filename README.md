# novilie Master
  Este es repositorio base del Proyecto de Sistema de Gestión Inmobiliaria: Novilie

  La manera de como actuar es la siguiente:
  - Los archivos que se editarán por cada desarrollador se subiran a su respectivo Brach según su categoría: BackEnd, FrontEnd y BBDD.
  - El Lider de Pruebas Funcionales deberá corroborar junto con el Gerente de Proyecto el correcto funcionamiento del Sistema con las nuevas funcionalidades subidas.
  - Después de que dichos archivos pasen la prueba, se procederá a transpasar los archivos aprovados del brach al master.
